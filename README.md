## Getting Started

This project has been created with 
node version: 22.19.0
nextJS version: 16

To install the project after cloneing it from git repo,

```bash
npm install
```
or you can use any package manager like pnpm, yarn or bun

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
